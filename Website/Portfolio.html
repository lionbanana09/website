This project is a task manager aplication. After completeing the python courses i wanted to apply me new found knowledge on a project that is more like what would be needed in a job. This aplication allows users to view, create, edit and delete tasks from an SQL database from a GUI. It checks due dates and will provide a pop up reminder if if less than 5 days away. It also provides the weather for the local area. I also made sure to build error handeling into the code so it wouldnt crash due to an unexpected or null input

This project was an attempt at creating a Ai chatbot. I created this around the ChatGPT hype as i wanted to get an understanting of how chatbots work and gain usefull skills. I tried two different aproaches to the chatbot following an online tutorial:
The first was made from scratch using transformers and the diabloGPT data. This chatbot worked but the replies were often unrelated to the question.
The next was made using the chatterbot library. This chatbot stayed on topic more often and can easily be trained by adding more training converstaions.
